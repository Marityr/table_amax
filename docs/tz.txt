алгоритм 

1) берем numberFix из входной таблицы 
2) делаем запрос в абцп 
3) абцп в ответ на запрос отдает массив, в котором в том числе есть поля brand и description (=пара «бренд + описание»)
4) из этого массива нас интересуют только те пары, в которых бренд (brand) входит в список подходящих брендов 
5) добавляем пары, прошедшие проверку на соответствие бренда, в выходную таблицу

api documentation https://www.abcp.ru/wiki/API.ABCP.Client

в ответ на запрос по numberFix=485208J025 система выдала массив из 10 позиций. 
В этом массиве нашему критерию отбора соответствуют только 2 - их и помещаем в таблицу
